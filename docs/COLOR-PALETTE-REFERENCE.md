## Color Usage Guidelines

### Text Hierarchy

```
Primary Text:    color['ink-base']   (#252d39) - Headings, important content
Secondary Text:  color['ink-light']  (#647492) - Body text, descriptions
Disabled Text:   color['grey-05']    (#8698ba) - Inactive/disabled content
```

### Background Hierarchy

```
Default:  color.offwhite      (#FEFEFE) - Main app background
Paper:    color.eggshellCream (#F5F2F0) - Card/paper surfaces
Elevated: color.white         (#ffffff) - Modals, popovers
```

### Action States

```
Default:  Base color
Hover:    *-hover variant (lighter/brighter)
Pressed:  *-pressed variant (darker)
Disabled: *-disabled variant (muted/faded)
```

## Color Naming Issues (Requires Attention)

### Misnamed Colors
These colors don't match their name descriptions:

| Variable | Hex | Name Says | Actually Is |
|----------|-----|-----------|-------------|
| `color.periwinkle` | `#7ebcff` | Purple/lavender | Light blue |
| `color.matteSeafoam` | `#de9b61` | Green | Tan/beige |
| `color.sunflower` | `#3b8386` | Yellow | Teal/cyan |
| `color.lavendar` | `#de4641` | Purple | Red/coral |
| `color.flatLime` | `#981d1a` | Lime green | Dark red |

**Recommendation**: Either rename the variables to match the actual colors, or change the hex values to match the names.

## Quick Color Reference (Most Common)

```typescript
// Copy-paste ready color references
const quickColors = {
  // Text
  textPrimary: color['ink-base'],      // #252d39
  textSecondary: color['ink-light'],   // #647492
  textDisabled: color['grey-05'],      // #8698ba

  // Backgrounds
  bgDefault: color.offwhite,           // #FEFEFE
  bgPaper: color.eggshellCream,        // #F5F2F0

  // Semantic
  error: color.crimson,                // #bf140a
  warning: color.burntorange,          // #ff9800
  info: color['blue-base'],            // #0a71d0
  success: color.applegreen,           // #41bf60

  // Brand
  primary: color.treegreen,            // #1C6F26
  secondary: color.walnut,             // #7b6748

  // Common
  black: color.ink,                    // #000000
  white: color.white,                  // #ffffff
  divider: color['grey-04'],           // #c8d2e6
}
```

## Export Format

All colors are exported from `colorGuide.ts`:

```typescript
import { color, styleguideColors, colorGuide } from './colorGuide';

// Access colors:
color.treegreen           // #1C6F26
color['blue-base']        // #0a71d0
styleguideColors.crimson  // #bf140a
colorGuide.walnut         // #7b6748
```
